### Objetivo
Criar um aplicativo web que explore a API construída na segunda atribuição. É esperado
a utilização do Angular, TypeScript, Angular CLI e SCSS.

## Instalação

```bash
$ npm install
```

## Executando o aplicativo

```bash
# development
$ npm run start

# production mode
$ npm run start --prod

# build production mode
$ npm run build --prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run e2e

```
